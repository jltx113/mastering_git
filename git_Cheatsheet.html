Git Quick Reference Sheet

Command Description
git init Initialize a new Git repository
git remote add origin <repo_url> Connect local repo to a remote repository
git branch -r List remote branches
git checkout -b main Create and switch to the 'main' branch
git add . Stage all changes
git commit -m "Initial commit" Commit staged changes with a message
git push -u origin main Push code to remote and set upstream
git branch -vv Show verbose branch information

Upstream branch: The remote branch that your local branch is tracking.

Verbose (-vv): Provides detailed branch tracking information.
